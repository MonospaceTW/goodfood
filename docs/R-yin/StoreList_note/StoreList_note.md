## 教訓：別在兩個不同環境下做事，A環境能跑不代表直接移植B環境不會出問題

嘿對，很雷，聽起來真的是這樣 (實際上也是)

在撰寫過程中，我曾經想過兩個版本都是使用 Vue-cli ，也是使用v2版本，那在使用上應該不會出什麼狀況吧

結果在移植的過程才發現兩邊的 package.json 中套件內容差異不小

最後理所當然的直接爆炸啦 (在重構專案中重構)
